More About Openpump

Openpump is an open-source hardware project initiated by Techealth at Swansea University. Our group has been doing a lot of research on the usability, interface and interaction design of infusion pumps and realised that no complete open-source infusion pump solution exists.

While there are some great open-source pump projects out there they tend to focus on one specific area, like the software or the physical design. We wanted to create a community that focuses on the whole solution, from the mechanical parts through to the user experience design.

Let's look at an example: If you're a software developer, you can try to reverse-engineer existing solutions or try to build simulations of how you think an infusion pump should work. However, if you would like to create software that you can test on a real pump, there is no complete open reference implementation you can build on that just works

We believe that you create the best work using the tools that you are most familiar with, so we try not to enforce a specific toolkit or workflow. However, if you're just getting started we do provide a number of tutorials and guides for tools and software we think will get you up and running the fastest.
